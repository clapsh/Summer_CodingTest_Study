## 📌 File | Commit | Pull Request 규칙
### File
```
사이트_문제번호_문제이름.확장자
```
### Commit
작업 내용의 종류는 다음과 같습니다.
- [Add] : 파일을 생성한 경우
- [Edit] : 파일을 수정한 경우
- [Delete] : 파일을 삭제한 경우
- [Fetch] : 브랜치 최신화를 하는 경우
```
git commit -m "[작업내용] 주차 | 주차별 번호 | 파일명"
```
### Pull Request
```
이름 | 주차 | 주차별 번호
```
